How darkwire Works

LoRa-Based Mesh Network

darkwire uses long-range LoRa radio technology to create a decentralized mesh network that bypasses traditional internet infrastructure entirely.

Multi-Hop Transaction Relay

Signed Bitcoin transactions hop from node to node until reaching a connected access point, ensuring your financial freedom even during blackouts.


Censorship Resistant

Transactions never touch monitored networks until the final exit node, preserving privacy and bypassing surveillance or censorship efforts.

Deployment Methods

darkwire nodes can be deployed in various ways to create resilient networks across challenging environments.

High-Altitude Balloon Node

Weather balloons equipped with darkwire nodes can provide wide-area coverage across controlled borders and restricted regions.

Maritime Buoy Node

Solar-powered buoys create mesh networks across water bodies to bridge communications between territories.

Portable Transceiver Node

Compact weatherproof enclosures house Arduino and LoRa modules for discrete deployment in urban or rural environments.

Technical Specifications

Hardware Requirements

  • Arduino UNO (or compatible microcontroller)
  • LoRa Transceiver Module (RA-02 or similar)
  • Power source (solar panels, batteries, or direct power)
  • Weatherproof enclosure for outdoor deployment
  • Only one node requires internet access to relay to the Bitcoin network
  • Range: Up to 10km per hop in rural areas, 1-2km in urban environments
